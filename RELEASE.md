## ✨ 功能清单

### 🔑 安全登录
- [x] 主密码保护，首次启动设置
- [x] 指纹 / 面部识别解锁
- [x] 应用切后台 30 秒自动锁定
- [x] 主密码修改（需验证旧密码或指纹）

### 📋 密码管理 (CRUD)
- [x] 手动添加：来源、账号、密码、备注、分类
- [x] 四列表格展示（序号 | 来源 | 账号 | 密码）
- [x] 搜索（来源 + 账号模糊匹配）
- [x] 分类筛选 + 排序（默认/名称/时间，升降序）
- [x] 编辑、删除（单条 + 批量）
- [x] 一键去重

### 📋 一键复制
- [x] 点击复制账号或密码到剪贴板
- [x] 密码复制后 30 秒自动清除
- [x] 密码默认脱敏显示（••••••••），点击切换明文

### 🔍 智能导入
- [x] 剪贴板监听，自动识别账号密码文本
- [x] 支持 6 种格式：中文键值对、分隔符、冒号、CSV、TSV、多列空格
- [x] 半屏弹窗确认，勾选后批量导入
- [x] 自动去重（相同来源 + 账号跳过）
- [x] CSV / JSON 文件导入

### 📤 数据导出
- [x] 加密导出为 CSV 或 JSON
- [x] 支持多选导出（编辑模式勾选）
- [x] 一键导出全部（设置页）
- [x] 导出前需验证主密码

### ↔️ 拖拽排序
- [x] 拖拽 ⋮⋮ 手柄调整顺序
- [x] ▲▼ 微调按钮
- [x] 松手自动保存

### 🔒 数据安全
- [x] AES-256-CBC 加密存储
- [x] PBKDF2 密钥派生（10000 次迭代）
- [x] 主密码仅存哈希（bcrypt 模式）
- [x] AES 密钥仅存内存，不落盘
- [x] 密码明文页禁止截屏/录屏
- [x] 剪贴板定时清除

### 🎨 用户体验
- [x] 多账号支持（用 `/` 分隔，下拉面板查看）
- [x] 账号智能格式化（邮箱完整、手机号脱敏）
- [x] 空状态引导提示
- [x] 编辑模式工具栏（全选/排序/导出/删除）
- [x] 自适应安全区域（状态栏不重叠）

## 🛠 技术栈

| 类别 | 选型 |
|------|------|

| 打包 | EAS Build |

## ⚠️ 注意事项

- **主密码无法找回**：不存服务器，忘记即丢失所有数据
- **导出文件含明文密码**：请妥善保管导出的 CSV/JSON 文件
- **建议定期导出备份**：防止手机丢失或损坏导致数据丢失
