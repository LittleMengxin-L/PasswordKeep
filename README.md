# 🔐 密码保险箱

一款安全、简洁的手机密码管理工具，帮你记住所有账号密码。

> ⚠️ 所有数据只存在你手机里，不上传到任何服务器。

---

## ✨ 能做什么

- 🔑 **一个主密码管所有** — 只需记住一个主密码，支持指纹/面部解锁
- 📋 **存密码** — 记录来源（哪个 App）、账号、密码，支持批量导入
- 🔍 **自动识别** — 复制一段含账号密码的文字，App 弹窗自动识别、一键导入
- 📋 **一键复制** — 点一下复制账号或密码，30 秒后自动清除剪贴板防泄露
- 🔒 **离开即锁** — 切到后台超时自动锁定，别人打不开
- 🛡 **AES-256 加密** — 所有密码加密存储，查看密码时禁止截屏
- 📂 **文件导入导出** — 支持 CSV/JSON 文件，换手机也能迁移
- ↔️ **拖拽排序** — 自定义密码排列顺序

## 🚀 怎么用

### 1. 设置主密码

首次打开输入一个 **至少 6 位** 的主密码，这是你**唯一需要记住**的密码。

### 2. 添加密码

三种方式任选：

| 方式 | 操作 |
|------|------|
| 手动添加 | 点右下角 **＋** → 填来源、账号、密码 → 保存 |
| 剪贴板识别 | 复制一段账号密码文字 → App 弹窗 → 勾选 → 导入 |
| 文件导入 | 导入 → 📂 选择文件 → 选 CSV/JSON → 解析 → 导入 |

### 3. 日常使用

- 列表中**点一下**某行 → 查看详情
- 点 **显示** 切换密码可见/隐藏
- 点 **📋** 复制账号或密码
- 点 **编辑** → 多选 → 批量删除/导出
- 点 **☰ 排序** → 拖拽调整顺序
- 设置 → **一键去重** 清理重复记录

## ❓ 常见问题

| 问题 | 答案 |
|------|------|
| 忘记主密码？ | ❌ 无法找回。主密码是唯一钥匙，请务必记牢 |
| 数据在哪？ | 📱 只在你的手机本地 |
| 换手机怎么迁移？ | 设置 → 导出全部 → CSV → 发到新手机 → 导入 |
| 指纹怎么用？ | 先密码登录一次 → 设置 → 开启指纹 |
| 会被盗吗？ | AES-256 加密，没有主密码解不开 |
